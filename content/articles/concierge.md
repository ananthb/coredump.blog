[Concierge](https://concierge.calculon.tech) is a messaging automation platform
for small businesses. It provides WhatsApp auto-replies, Instagram DM
auto-replies, and embeddable lead capture forms with zero setup required.

Built with Rust and WebAssembly on Cloudflare Workers, it integrates with
Meta's WhatsApp Business API and Instagram Graph API, using Cloudflare AI, D1,
and KV for storage and intelligence.
