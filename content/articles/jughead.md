[Jughead](https://github.com/ananthb/jughead) is a clean, minimal blog theme
for Hugo with two color schemes (ink and earth), auto light/dark mode, Giscus
comments, MermaidJS diagrams, and KaTeX math rendering.

It uses Pico CSS as a lightweight base, layers custom theming via CSS variables,
and supports SVG favicons, reading time indicators, and offline caching with a
service worker.
